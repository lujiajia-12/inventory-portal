## 二、每日操作流程（仅工作日 周一至周五）

```
时间          谁                操作
────────────────────────────────────────────────────
08:30     系统自动          飞书消息提醒各仓位负责人更新库存
                            （周末自动跳过，不打扰）

08:30~    各仓位负责人      打开飞书表格 → 切换到自己的视图
09:30                      找到自己负责的仓库列 → 改库存数字
                           改完自动保存，无需额外操作

09:45     系统自动          检查谁还没更新 → 发给段慧琴
                            （周末自动跳过）

09:45~    段慧琴           看未更新清单 → 电话提醒
09:55

10:00     系统自动          ★ 快照归档 ★
                           复制库存总表全部数据 → 写入快照日志表
                           推送库存日报卡片到销售群

10:00后   销售             在销售群里看到日报卡片
                           想查详情 → 打开表格看"销售看板"视图

全天      销售             随时打开飞书看"销售看板"
                           搜索产品→查各仓位库存
```

> ⚠️ **周末**：所有定时任务自动跳过（脚本内置了工作日判断），无人需要加班。周五的快照数据保留到周一。

## 三、各角色操作指南

### 3.1 仓管员/供应商（每日操作，30秒搞定）

```
第1步: 打开飞书 → 找到「70迈库存表2026-6-22」
第2步: 切换到自己的视图（如"永惠仓 — 库存录入"）
第3步: 找到要更新的产品行
第4步: 在自己那列的单元格里，把库存数字改成今天的实际数字
第5步: 改完直接关闭，飞书自动保存。完成。
```

**注意**:
- 你只能看到和修改自己仓的列，其他仓的列是被隐藏的
- 如果当天库存无变化，也请点击一下数字再确认（确保更新人有记录）
- 如有异常情况（在途、退货等），在「备注」列选择或填写说明

### 3.2 销售（查库存，10秒搞定）

```
方式1（推荐）: 
  打开飞书表格 → 切换到"销售 — 库存看板"视图
  → 顶部搜索框输入产品名/料号 → 看到四个仓位的库存总和

方式2:
  在销售群里翻看每天10:00的日报卡片 → 点击链接跳转

方式3（未来可用机器人）:
  群内 @库存助手 查304螺栓
  → 机器人返回各仓位库存明细
```

### 3.3 段慧琴（系统管理员）

```
每日工作:
  08:30  收到系统推送的更新提醒（查看即可）
  09:45  收到未更新清单 → 电话跟进
  10:00  收到快照完成通知 → 检查日报是否有异常数据

其他:
  新增产品 → 在库存总表末尾添加新行 → 填写基础信息
  停产产品 → 将库存改为0 → 备注写"停产"
  改基础信息 → 直接在库存总表中修改
  查历史 → 在快照日志表中搜索
```

## 六、常见问题

**Q: 某仓位忘记更新怎么办？**
> 09:45系统检测到未更新 → 通知段慧琴 → 段慧琴电话跟进。快照时如未更新会沿用昨日数据。

**Q: 周末怎么处理？**
> 快照照常生成（数据沿用周五）。可以设置工作日提醒，周末跳过催更。

**Q: 新增产品怎么做？**
> 段慧琴在库存总表末尾加行 → 填基础信息（产品线到产品模式）→ 仓位库存留0 → 各仓下次更新时填写。

**Q: 填错了数字能改吗？**
> 可以，直接在库存总表中改回正确数字。历史快照不动，保证审计完整性。

**Q: 快照日志表能存多久？**
> 857产品×365天=31万行/年。飞书标准版单表10万行上限，建议每年归档一次（导出CSV备份后清理）。

**Q: 想查某个产品3个月的库存走势？**
> 打开快照日志表 → 搜索料号 → 就能看到每天的快照记录。
