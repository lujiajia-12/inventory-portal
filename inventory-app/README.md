# 📦 仓库盘点小程序

基于飞书多维表格数据的移动端仓库盘点工具，支持物料编码搜索、商品名称模糊搜索、69码扫码快速盘点。

## 功能

- **物料编码后4位查找** — 输入编码后4位数字快速定位商品
- **商品名称模糊搜索** — 输入关键词即时匹配
- **手机扫码69码** — 调用摄像头扫描 EAN-13 条形码自动查找
- **盘点明细** — 展示商品名称、物料编码、在库数量、69码
- **备货区/库存区盘点** — 分别填写两个区域的盘点数量，自动计算差异
- **一键提交** — 提交后自动跳转下一个待盘商品
- **导出结果** — 支持 CSV 格式导出盘点记录
- **进度追踪** — 实时显示已盘/剩余统计

## 使用方法

### 桌面端
直接用浏览器打开 `index.html` 即可使用。

### 移动端（推荐）
1. 将 `inventory-app` 文件夹部署到 Web 服务器
2. 手机浏览器打开对应地址
3. 也可通过局域网访问：在本机用 `npx serve inventory-app` 启动后手机访问

> 扫码功能需要 HTTPS 或 localhost 环境才能调用摄像头。

### 操作流程

1. 选择搜索方式：**物料编码** / **商品名称** / **扫码**
2. 输入编码后4位或商品名称关键词 → 显示搜索结果
3. 点击商品进入盘点明细
4. 填写「备货区数量」和「库存区数量」
5. 查看盘点合计与在库数量的差异
6. 点击 **"✓ 一键提交，盘下一个"** → 自动跳转下一个待盘商品
7. 全部盘点完毕后可导出 CSV 结果

### 快捷键

| 快捷键 | 作用 |
|--------|------|
| `Enter` | 搜索框中触发搜索 |
| `Ctrl + Enter` | 盘点明细页快速提交 |

## 数据对接

当前使用 Mock 数据（`js/data.js`），包含 20 条示例商品数据。对接飞书多维表格时：

1. 修改 `js/data.js` 中的查询函数，替换为飞书 API 调用
2. 保持函数签名不变（参数和返回值格式一致）
3. 其他模块无需改动

飞书 API 对接示例参考：
- 飞书多维表格 API: https://open.feishu.cn/document/server-docs/docs/bitable-v1/overview

### Mock 数据结构

```javascript
{
  id: 1,                          // 唯一标识
  name: '纯牛奶 250ml×24盒',      // 商品名称
  code: 'WL-2024-0001',           // 物料编码
  stock: 120,                     // 在库数量
  prepArea: '',                   // 备货区数量（盘点时填写）
  storageArea: '',                // 库存区数量（盘点时填写）
  barcode: '6901234567890',       // 商品69码 (EAN-13)
  unit: '箱'                      // 单位
}
```

## 技术栈

- 纯 HTML5 + CSS3 + Vanilla JavaScript
- [html5-qrcode](https://github.com/mebjas/html5-qrcode) — 摄像头条码扫描
- LocalStorage — 本地数据持久化
